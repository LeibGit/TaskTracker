# Task Tracker App - Key Functionality

- [ ] View all tasks (GET /tasks)
- [ ] Add new task (POST /tasks)
- [ ] Toggle task completed status (PUT /tasks/:id)
- [ ] Delete task (DELETE /tasks/:id)
- [ ] Backend stores tasks (Python list, FastAPI endpoints)
